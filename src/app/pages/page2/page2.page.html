Definición de componentes, propiedades, atributos de componentes y eventos

PÁGINA 2: DEFINICIONES.

Componentes ¿Qué es un componente? La mejor forma de definir un componente es referirnos a una función, ya que se trata de código que podemos reutilizar aplicándolo a distintos contextos. Propiedades y atributos de un componente ¿Para qué sirven? Las propiedades o atributos son los fragmentos de código necesarios para formar un componente, es decir, podriamos tener un componente el cual nos muestre un mensaje de texto utilizando una propiedad la cual sea un String y podriamos tener otra propiedad la cual sea una variable tipo Integer para mostrar un número. Eventos ¿Cuando suceden? Para comprender lo que es un evento, debemos pensar en la acción que se ejecuta cuando cambiamos de un tab a otro, ya que un evento es el funcionamiento en segundo plano de esa acción.
